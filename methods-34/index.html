chapter 1 | algebra and functions

transformation

translation of \(h\) units in the direction of the x-axis

$$f(x-h)$$

translation of \(k\) units in the direction of the y-axis

$$f(x)+k$$

dilation by a scale factor \(a\) from the y-axis

$$f(\frac{x}{a})\qquad\qquad\begin{bmatrix}a&0\\0&1\end{bmatrix}$$

dilation by a scale factor \(b\) from the x-axis

$$kf(x)\qquad\qquad\begin{bmatrix}1&0\\0&b\end{bmatrix}$$

reflection in the x-axis

$$-f(x)\qquad\qquad\begin{bmatrix}1&0\\0&-1\end{bmatrix}$$

reflection in the y-axis

$$f(-x)\qquad\qquad\begin{bmatrix}-1&0\\0&1\end{bmatrix}$$

polynomials

remainder theorem: the remainder when \(P(x)\) is divide by \((ax+b)\) is \(P(-\frac{b}{a})\)

factor theorem: \(P(x)\) has factor \((ax+b)\) if \(P(-\frac{b}{a})=0\)

quadratic functions

the quadratic equation \(ax^2+bx+c\) has general solutions

$$x=\frac{-b\pm\sqrt{b^2-4ac}}{2a}$$

the discriminant: \(\Delta=b^2-4ac\)

if \(\Delta\gt 0\), the quadratic has two real solutions

if \(\Delta=0\), the quadratic has one real solution

if \(\Delta\lt 0\), the quadratic has no real solutions

index laws

$$x^a\times x^b=x^{a+b}$$

$$x^a\div x^b=x^{a-b}$$

$$(x^a)^b=x^{ab}$$

$$(xy)^a=x^ay^a$$

$$(\frac{x}{y})^a=\frac{x^a}{y^a}$$

$$x^0=1$$

logarithm laws

$$\log_x(a)+\log_x(b)=\log_x(ab)$$

$$\log_x(a)-\log_x(b)=\log_x(\frac{a}{b})$$

$$\log_x(a^b)=b\log_x(a)$$

$$\log_x(\frac{1}{a})=-\log_x(a)$$

$$\log_x(x)=1$$

$$\log_x(1)=0$$

$$\log_a(b)=\frac{\log_e(b)}{\log_e(a)}=\frac{\log_{10}(b)}{\log_{10}(b)}$$

symmetry properties of the circular functions

\(\sin(\pi-\theta)=\sin(\theta)\) \(\sin(\pi+\theta)=-\sin(\theta)\)
\(\cos(\pi-\theta)=-\cos(\theta)\) \(\cos(\pi+\theta)=-\cos(\theta)\)
\(\tan(\pi-\theta)=-\tan(\theta)\) \(\tan(\pi+\theta)=\tan(\theta)\)
\(\sin(2\pi-\theta)=-\sin(\theta)\) \(\sin(-\theta)=-\sin(\theta)\)
\(\cos(2\pi-\theta)=\cos(\theta)\) \(\cos(-\theta)=\cos(\theta)\)
\(\tan(2\pi-\theta)=-\tan(\theta)\) \(\tan(-\theta)=-\tan(\theta)\)

complementary angles of the circular functions

\(\sin(\frac{\pi}{2}-\theta)=\cos(\theta)\) \(\sin(\frac{\pi}{2}+\theta)=\cos(\theta)\)
\(\cos(\frac{\pi}{2}-\theta)=\sin(\theta)\) \(\cos(\frac{\pi}{2}+\theta)=-\sin(\theta)\)

exact values of the circular functions

\(\sin(0)=0\) \(\cos(0)=1\) \(\tan(0)=0\)
\(\sin(\frac{\pi}{6})=\frac{1}{2}\) \(\cos(\frac{\pi}{6})=\frac{\sqrt{3}}{2}\) \(\tan(\frac{\pi}{6})=\frac{1}{\sqrt{3}}\)
\(\sin(\frac{\pi}{4})=\frac{1}{\sqrt{2}}\) \(\cos(\frac{\pi}{4})=\frac{1}{\sqrt{2}}\) \(\tan(\frac{\pi}{4})=1\)
\(\sin(\frac{\pi}{3})=\frac{\sqrt{3}}{2}\) \(\cos(\frac{\pi}{3})=\frac{1}{2}\) \(\tan(\frac{\pi}{3})=\sqrt{3}\)
\(\sin(\frac{\pi}{2})=1\) \(\cos(\frac{\pi}{2})=0\) \(\tan(\frac{\pi}{3})=\text{undefine}\)
\(\sin(\pi)=0\) \(\cos(\pi)=-1\) \(\tan(\pi)=0\)

power functions

\(y=x^2\) is the rule for a quadratic function and its graph is a parabola

\(y=x^3\) is the rule for a cubic function and its graph is a cubic graph

\(y=x^4\) is the rule of a quartic function and its graph is a quartic graph

\(y=x^{\frac{1}{2}}\) or \(y=\sqrt{x}\) is the rule for a square root function and its graph is a square root graph

the graph of a function with rule \(y=x^{-1}\) \(y=\frac{1}{x}\) is a hyperbola

the graph of a function with rule \(y=x^{-2}\) or \(y=\frac{1}{x^2}\) is a truncus

factorised form

when the equation of a polynomial function is in factorised form so that each factor is linear (in the form \(y=ax+b\)), it is very easy to read off the x-intercepts by simply finding the x-value for which the expressions inside the brackets equal zero. we then use the x- and y-intercepts to sketch the graph

exponential functions

an exponential function is defined as having a rule

$$y=a^x,a\in R\setminus\{1\}$$

exponential graphs of the form above have the following properties in common:

all pass through \(0, 1\) as \(a^0=1\)

all pass through \((1, a)\) as \(a^1=a\)

all have horizontal asymptote at \(y=0\)

the domain is \(R\) and the range is \(R^+\)

logarithmic functions

a logarithmic function is defined as having a rule

$$y=\log_ax,a\in R^+\setminus\{1\}$$

logarithmic graphs of the form above have the following properties in common:

all pass through \(1, 0\) as \(log_a1=0\) (as \(a^0=1\))

all pass through \((a, 1)\) as \(log_aa=1\) (as \(a^1=a\))

all have vertical asymptote at \(x=0\) (as we can only take the log of a positive number)

the domain is \(R^+\) and the range is \(R\)

sine and cosine graphs

the domain for both the sine function and the cosine function is \(R\)

for a graph with equation of the form \(y=a\sin(nx)\) or \(y=a\cos(nx)\), the amplitude is \(a\) and the period \(P=\frac{2\pi}{\lvert n\rvert}\)

tangent graphs

\(\tan(x)=\frac{\sin(x)}{\cos(x)}\), so the function does not exist when \(\cos(x)=0\)

there are asymptote wherever \(\cos(x)=0\)

for a graphs with equation of the form \(y=\tan(nx)\), the period \(p=\frac{\pi}{\lvert n\rvert}\)

composite functions

\(f(g(x))\) are \(g(f(x))\) are composite functions â€“ there is a function within another function. they can also be written as \(f\circ g(x)\) and \(g\circ f(x)\) respectively. the largest possible domain is the domain of the inner function. for a composite function to exist, the range of the inner function must be a subset or equal to the domain of the outer function

inverse functions

for an inverse function to exist, the original function must be one to one. to find the rule for an inverse function, we swap \(x\) and \(y\) in the original rule and then make \(y\) the subject so that \(y\) is then equal to \(f^{-1}(x)\)

the domain of the inverse will therefore be the range of the original and vice versa. to draw \(f^{-1}(x)\), we reflect the graph of \(f(x)\) in the line \(y=x\)
