| $f(x)$       | $f'(x)$              |
| ------------ | -------------------- |
| $x^n$        | $nx^{n-1}$           |
| $(ax+b)^n$   | $an(ax+b)^{n-1}$     |
| $e^{f(x)}$   | $f'(x)e^{f(x)}$      |
| $\log_ef(x)$ | $\frac{f'(x)}{f(x)}$ |
| $\sin{f(x)}$ | $f'(x)\cos{f(x)}$    |
| $\cos{f(x)}$ | $-f'(x)\sin{f(x)}$   |
| $\tan{f(x)}$ | $f'(x)\sec^2{f(x)}$  |
